C Programming Language

Introduction

C programming is one of the most important and widely used programming languages in the world. It was developed in 1972 by Dennis Ritchie at Bell Laboratories.

What is C?

C is a general-purpose, procedural programming language used to develop system software, application software, and embedded systems.

Why Learn C?

  • It is the foundation of many modern languages like C++, Java, and Python
  • Helps understand low-level memory management
  • Widely used in operating systems, compilers, and embedded systems

1. Variables and Data Types

A variable is a name used to store data in memory. Data types specify what type of data the variable will store.


#include <stdio.h>

int main() {
    int age = 20;        // integer
    float price = 99.5;  // decimal number
    char grade = 'A';    // character

    printf("Age: %d\n", age);
    printf("Price: %.2f\n", price);
    printf("Grade: %c\n", grade);

    return 0;
}

1. Variable (Definition)

A variable is a name given to a memory location used to store data in a program. Its value can change during program execution.

2. Declaration of Variable

Definition: Variable declaration is the process of specifying the data type and name of a variable.

Syntax:

data_type variable_name;

Example:

int age;

3. Initialization of Variable

Definition: Initialization means assigning a value to a variable at the time of declaration.

Example:

int age = 20;

4. Rules for Naming Variables

Definition: Variable naming rules define how identifiers should be written in C.

  • Must begin with a letter or underscore
  • Cannot start with a digit
  • Cannot use keywords
  • No spaces allowed
  • Case-sensitive

5. Data Types (Definition)

A data type defines the type of data a variable can store, the size of memory it occupies, and the operations that can be performed on it.

6. Types of Data Types in C

A. Basic (Primary) Data Types

Definition: Basic data types are fundamental data types provided by C to store simple values.

  • int → stores integers
  • float → stores decimal numbers
  • double → stores large decimal numbers
  • char → stores a single character

B. Derived Data Types

Definition: Derived data types are formed from basic data types.

  • Arrays
  • Pointers
  • Functions

C. User-Defined Data Types

Definition: User-defined data types are created by the programmer to define custom data structures.

  • struct
  • union
  • enum
  • typedef

7. Size of Data Types

Definition: The size of a data type determines how much memory it occupies.

  • char → 1 byte
  • int → 2 or 4 bytes
  • float → 4 bytes
  • double → 8 bytes

8. Type Modifiers

Definition: Type modifiers are used to modify the size or range of basic data types.

  • short
  • long
  • signed
  • unsigned

Example:

unsigned int x;
long int y;

9. Constants

Definition: A constant is a fixed value that cannot be changed during program execution.

Example:

const int x = 10;

10. Keywords

Definition: Keywords are reserved words in C that have predefined meanings and cannot be used as variable names.

Examples: int, float, if, return, while

11. Format Specifiers

Definition: Format specifiers are used in input/output functions to define the type of data being read or printed.

  • %d → integer
  • %f → float
  • %c → character
  • %lf → double

12. Type Conversion

Definition: Type conversion is the process of converting one data type into another.

(a) Implicit Conversion

Automatically done by compiler.

int a = 5;
float b = a;   // int → float

(b) Explicit Conversion (Type Casting)

Manually done by programmer.

float a = 5.5;
int b = (int)a;

13. Storage Classes (Basic Idea)

Definition: Storage classes define the scope, lifetime, and visibility of variables.

  • auto
  • register
  • static
  • extern

14. Example Program

#include <stdio.h>

int main() {
    int age = 20;
    float marks = 85.5;
    char grade = 'A';

    printf("Age: %d\n", age);
    printf("Marks: %f\n", marks);

    return 0;
}

2. Operators

Operators are used to perform operations on variables.

  • Arithmetic → + - * / %
  • Relational → > < == !=
  • Logical → && || !

#include <stdio.h>
int main() {
    int a = 10, b = 5;

    printf("Sum = %d\n", a + b);
    printf("Is a > b? %d\n", a > b);

    return 0;
}

1. Arithmetic Operators

Used for mathematical calculations.

Operator Meaning
+ Addition
- Subtraction
* Multiplication
/ Division
% Modulus (remainder) 
#include <stdio.h>

Used to compare two values. Result is true (1) or false (0).

Operator Meaning

!= Not equal

3. Logical Operators

Used to combine conditions.

Operator Meaning
&& Logical AND
! Logical NOT 
#include <stdio.h>
int main() {
    int a = 5, b = 10;

    if (a < 10 && b > 5) {
        printf("Both conditions are true\n");
    }

    if (a > 10 || b > 5) {
        printf("At least one condition is true\n");
    }

    if (!(a == b)) {
        printf("a is not equal to b\n");
    }

    return 0;
}

Used to assign values to variables.

Operator Meaning

5. Increment and Decrement Operators

Used to increase or decrease value by 1.

Operator Meaning
++ Increment
-- Decrement

Types:

  • Pre-increment (++a)
  • Post-increment (a++)

#include <stdio.h>
int main() {
    int a = 5;

    printf("%d\n", ++a); // 6 (first increase, then print)
    printf("%d\n", a++); // 6 (print, then increase)
    printf("%d\n", a);   // 7

    return 0;
}

3. Control Structures

They control the flow of execution.

if-else Example:


#include <stdio.h>

int main() {
    int age = 18;

    if (age >= 18) {
        printf("You can vote");
    } else {
        printf("You cannot vote");
    }

    return 0;
}

Loop Example (for loop):


#include <stdio.h>

int main() {
    for(int i = 1; i <= 5; i++) {
        printf("%d\n", i);
    }
    return 0;
}

Types of Control Structures in C

There are 3 main types of control structures:

  • Selection (Decision-Making)
  • Iteration (Looping)
  • Jump Statements

1. Selection Statements (Decision Making)

Used to make decisions based on conditions.

(a) if Statement

The if statement is used to execute a block of code only when a specified condition is true. 


if (condition) {
    // code
}
      

int age = 18;

if (age >= 18) {
    printf("You can vote");
}

(b) if-else Statement

The if-else statement is used to execute one block of code if the condition is true and another block if the condition is false.


if (condition) {
    // true block
} else {
    // false block
}
      

if (age >= 18) {
    printf("Adult");
} else {
    printf("Minor");
}

(c) else-if Ladder

The else-if ladder is used to check multiple conditions one by one and execute the block of code corresponding to the first true condition.


if (condition1) {
    // code
} else if (condition2) {
    // code
} else {
    // default
}
      

int marks = 75;

if (marks >= 90) {
    printf("Grade A");
} else if (marks >= 60) {
    printf("Grade B");
} else {
    printf("Fail");
}

(d) switch Statement

The switch statement is used to select one block of code from multiple options based on the value of an expression.


switch (expression) {
    case value1:
        // code
        break;
    case value2:
        // code
        break;
    default:
        // code
}
      

int day = 2;

switch(day) {
    case 1: printf("Monday"); break;
    case 2: printf("Tuesday"); break;
    default: printf("Invalid");
}

👉 break stops execution from falling into next cases.

2. Iteration Statements (Loops)

Used to repeat a block of code multiple times.

(a) for Loop

The for loop is used to execute a block of code repeatedly for a fixed number of times.


for (initialization; condition; update) {
    // code
}
      

for (int i = 1; i <= 5; i++) {
    printf("%d\n", i);
}

(b) while Loop

The while loop is used to repeatedly execute a block of code as long as the given condition is true.


while (condition) {
    // code
}
      

int i = 1;

while (i <= 5) {
    printf("%d\n", i);
    i++;
}

(c) do-while Loop

The do-while loop is used to execute a block of code at least once, and then repeat it as long as the condition is true.


do {
    // code
} while (condition);
      

int i = 1;

do {
    printf("%d\n", i);
    i++;
} while (i <= 5);

3. Jump Statements

Used to transfer control immediately.

(a) break

The break statement is used to immediately terminate a loop or switch statement and transfer control to the next statement.


for (int i = 1; i <= 5; i++) {
    if (i == 3)
        break;
    printf("%d\n", i);
}

(b) continue

The continue statement is used to skip the current iteration of a loop and continue with the next iteration.


for (int i = 1; i <= 5; i++) {
    if (i == 3)
        continue;
    printf("%d\n", i);
}

(c) goto

The goto statement is used to transfer control to a labeled statement within the same function.


goto label;

label:
    printf("Hello");

(d) return

The return statement is used to exit from a function and optionally return a value to the calling function.


return 0;

4. Functions

A function is a reusable block of code.


#include <stdio.h>

// function definition
int add(int x, int y) {
    return x + y;
}

int main() {
    int result = add(4, 6); // function call
    printf("Sum = %d", result);

    return 0;
}

1. What is a Function?

A function is a block of code that performs a specific task and can be called whenever needed.

👉 Instead of writing the same code again and again, you define it once and reuse it.

2. Basic Structure of a Function

return_type function_name(parameters) {
    // body of function
    return value;
}

Example:

int add(int a, int b) {
    return a + b;
}

3. Parts of a Function

1. Function Declaration (Prototype)

int add(int, int);

2. Function Definition

int add(int a, int b) {
    return a + b;
}

3. Function Call

int result = add(5, 3);

Complete Example

#include <stdio.h>

// Function declaration
int add(int, int);

int main() {
    int result;
    result = add(5, 3);  // function call
    printf("Sum = %d\n", result);
    return 0;
}

// Function definition
int add(int a, int b) {
    return a + b;
}

Types of Functions in C

1. Library Functions

Predefined functions provided by C.

  • printf()
  • scanf()
  • sqrt()

2. User-defined Functions

Functions created by the programmer.

Types Based on Arguments & Return Value

1. No Arguments, No Return Value

#include <stdio.h>

void greet() {
    printf("Hello!\n");
}

int main() {
    greet();
    return 0;
}

2. Arguments, No Return Value

#include <stdio.h>

void greet(char name[]) {
    printf("Hello %s\n", name);
}

int main() {
    greet("Rahul");
    return 0;
}

3. No Arguments, With Return Value

#include <stdio.h>

int getNumber() {
    return 10;
}

int main() {
    int num = getNumber();
    printf("%d\n", num);
    return 0;
}

4. Arguments and Return Value (Most Important)

#include <stdio.h>

int multiply(int a, int b) {
    return a * b;
}

int main() {
    int result = multiply(4, 5);
    printf("Result = %d\n", result);
    return 0;
}

Call by Value

C uses call by value, meaning a copy of variables is passed.

#include <stdio.h>

void change(int x) {
    x = 100;
}

int main() {
    int a = 10;
    change(a);
    printf("%d\n", a);  // still 10
    return 0;
}

Call by Reference (Using Pointers)

Used to modify original values.

#include <stdio.h>

void change(int *x) {
    *x = 100;
}

int main() {
    int a = 10;
    change(&a);
    printf("%d\n", a);  // now 100
    return 0;
}

Recursive Function

A function that calls itself.

Example: Factorial

#include <stdio.h>

int factorial(int n) {
    if (n == 0)
        return 1;
    else
        return n * factorial(n - 1);
}

int main() {
    printf("Factorial = %d\n", factorial(5));
    return 0;
}

Advantages of Functions

  • ✔ Code reusability
  • ✔ Easy debugging
  • ✔ Better readability
  • ✔ Modular programming

Important Notes

  • Every C program must have main() function
  • Functions can return only one value (but can use pointers/structures for more)
  • Function names should be meaningful
  • Parameters are optional

1. Arrays in C

What is an Array?

An array is a collection of elements of the same data type stored in contiguous memory locations.

Declaration of Array

data_type array_name[size];

int arr[5];

Initialization

int arr[5] = {10, 20, 30, 40, 50};
int arr[] = {1, 2, 3};

Accessing Elements

#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};

    printf("%d\n", arr[0]); // 10
    printf("%d\n", arr[1]); // 20
    printf("%d\n", arr[2]); // 30

    return 0;
}

Using Loop with Array

#include <stdio.h>

int main() {
    int arr[5] = {1, 2, 3, 4, 5};

    for(int i = 0; i < 5; i++) {
        printf("%d ", arr[i]);
    }

    return 0;
}

Types of Arrays

1. One-Dimensional Array

int arr[5];

2. Two-Dimensional Array (Matrix)

#include <stdio.h>

int main() {
    int arr[2][2] = {{1, 2}, {3, 4}};

    for(int i = 0; i < 2; i++) {
        for(int j = 0; j < 2; j++) {
            printf("%d ", arr[i][j]);
        }
        printf("\n");
    }

    return 0;
}

2. Pointers in C

What is a Pointer?

A pointer is a variable that stores the address of another variable.

Declaration

data_type *pointer_name;
int *p;

Initialization

int a = 10;
int *p = &a;
  • &a → address of a
  • p → stores address
  • *p → value at that address

Example

#include <stdio.h>

int main() {
    int a = 10;
    int *p = &a;

    printf("Value of a: %d\n", a);
    printf("Address of a: %p\n", &a);
    printf("Pointer p stores: %p\n", p);
    printf("Value using pointer: %d\n", *p);

    return 0;
}

Pointer Arithmetic

#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};
    int *p = arr;

    printf("%d\n", *p);       // 10
    printf("%d\n", *(p + 1)); // 20
    printf("%d\n", *(p + 2)); // 30

    return 0;
}

3. Relationship Between Arrays and Pointers

👉 Array name itself is a pointer to the first element.
arr is same as &arr[0]. 

#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};

    printf("%d\n", *arr);       // 10
    printf("%d\n", *(arr + 1)); // 20
    printf("%d\n", *(arr + 2)); // 30

    return 0;
}

Difference Between Array and Pointer

Array Pointer
Fixed size Dynamic
Cannot change address Can change address
Stores multiple values Stores address

7. Dynamic Memory Allocation (Pointers)

#include <stdio.h>
#include <stdlib.h>
int main() {
    int *p;

    for(int i = 0; i < 3; i++) {
        printf("%d ", p[i]);

    return 0;
}

6. Structures

Structures group different types of data together.


#include <stdio.h>

struct Student {
    int id;
    char name[20];
    float marks;
};

int main() {
    struct Student s1 = {1, "Rahul", 88.5};

    printf("ID: %d\n", s1.id);
    printf("Name: %s\n", s1.name);
    printf("Marks: %.2f\n", s1.marks);

    return 0;
}

1. What is a Structure in C?

A structure (struct) is a user-defined data type that allows grouping different types of variables.

  • roll (int)
  • name (string)
  • marks (float)

2. Syntax of Structure

struct structure_name {
    data_type member1;
    data_type member2;
};

3. Declaring Structure

struct Student {
    int roll;
    char name[50];
    float marks;
};

4. Creating Variables

struct Student s1, s2;

5. Accessing Members

s1.roll = 1;
s1.marks = 90.5;

6. Complete Example

#include <stdio.h>
#include <string.h>

struct Student {
    int roll;
    char name[50];
    float marks;
};

int main() {
    struct Student s1;

    s1.roll = 1;
    strcpy(s1.name, "Rahul");
    s1.marks = 88.5;

    printf("Roll: %d\n", s1.roll);
    printf("Name: %s\n", s1.name);
    printf("Marks: %.2f\n", s1.marks);

    return 0;
}

7. Initialization

struct Student s1 = {1, "Amit", 92.5};

8. Array of Structures

struct Student s[3] = {
    {1, 80.5},
    {2, 85.0},
    {3, 90.0}
};

9. Pointer to Structure

struct Student *p = &s1;

By Value:

void display(struct Student s)

void update(struct Student *s)
