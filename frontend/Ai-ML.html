Day 3

Measures of Central Tendency

1. Mean (Average)

Definition: Mean is the sum of all values divided by total number of values.

Formula: Mean = Σx / n

Example: Data: 10, 20, 30 → Mean = (10+20+30)/3 = 20

import numpy as np
data = [10,20,30]
print(np.mean(data))

2. Median

Definition: Middle value after sorting

Example: [10, 20, 30] → 20

3. Mode

Definition: Most frequently occurring value

Why Important in AI?

  • Data understanding
  • Outlier detection

Day 4

Variance and Standard Deviation

Variance

Definition: Measures how far data points are from mean.

Formula: Variance = Σ(x − μ)² / n

Standard Deviation

Formula: Std = √Variance

Example

Data: [10, 20, 30]

  • Mean = 20
  • Differences = [-10, 0, 10]
  • Squared = [100, 0, 100]
  • Variance = 66.67

Code

import numpy as np

data = [10,20,30]

print(np.var(data))
print(np.std(data))

Importance in AI

  • Detect noise
  • Normalize data
