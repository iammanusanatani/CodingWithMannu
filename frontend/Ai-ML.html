Day 3

Measures of Central Tendency

1. Mean (Average)

Definition: Mean is the sum of all values divided by total number of values.

Formula: Mean = Σx / n

Example: Data: 10, 20, 30 → Mean = (10+20+30)/3 = 20

import numpy as np
data = [10,20,30]
print(np.mean(data))

2. Median

Definition: Middle value after sorting

Example: [10, 20, 30] → 20

3. Mode

Definition: Most frequently occurring value

Why Important in AI?

  • Data understanding
  • Outlier detection

Day 4

Variance and Standard Deviation

Variance

Definition: Measures how far data points are from mean.

Formula: Variance = Σ(x − μ)² / n

Standard Deviation

Formula: Std = √Variance

Example

Data: [10, 20, 30]

  • Mean = 20
  • Differences = [-10, 0, 10]
  • Squared = [100, 0, 100]
  • Variance = 66.67

Code

import numpy as np

data = [10,20,30]

print(np.var(data))
print(np.std(data))

Importance in AI

  • Detect noise
  • Normalize data

Python is a high-level, interpreted programming language designed to be simple, readable, and powerful. It is widely used in Artificial Intelligence, Data Science, and Automation.

Python is popular in AI because it is easy to learn, has a large ecosystem of libraries, and allows fast development of models and applications.

Data types define the kind of value stored in a variable:

Neural Networks Basics

What is a Neural Network?

A Neural Network is a computing system inspired by the human brain. It learns patterns from data using layers of interconnected nodes called neurons.

Structure

  • Output Layer → gives prediction

Why it matters

It helps machines learn complex patterns like image recognition and speech understanding.

Day 37

Forward & Backpropagation

What is Forward Propagation?

Forward propagation is the process where input data moves through the neural network to generate an output (prediction).

What is Backpropagation?

Backpropagation is the learning process where the model adjusts weights based on error to improve accuracy.

Simple Idea

Forward = prediction
Backward = learning from mistakes

Why important?

This is how deep learning models improve automatically.

Day 38

Activation Functions

What are Activation Functions?

Activation functions decide whether a neuron should be activated or not based on input.

Common Functions

  • ReLU → removes negative values
  • Sigmoid → outputs probability (0 to 1)
  • Tanh → outputs between -1 and 1

Why important?

They introduce non-linearity, allowing neural networks to learn complex patterns.
